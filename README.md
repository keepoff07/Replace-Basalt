# Replace-Basalt
玄武岩の上下面をいい感じに

バニラの玄武岩を直視できない人向けに、  
上下面をかまどのテクスチャに置き換えたリソースパックです。

## スクリーンショット
バニラテクスチャの玄武岩もいるので一応閲覧注意で。

## 動作確認
Minecraft JavaEdition ver1.19.4

今後マイクラのアプデでリソースパックに警告出ても、  
玄武岩のテクスチャしか変えてないので基本そのまま使えるはずです。
